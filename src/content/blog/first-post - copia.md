# Síntoma

El sensor aparecía conectado en la consola, pero las firmas no se actualizaban correctamente.

# Diagnóstico

- Se validó conectividad hacia Internet.
- Se verificó acceso a los servidores de actualización.
- Se revisaron logs del sensor.

# Solución

Después de revisar la configuración y la conectividad, el problema quedó resuelto ajustando los parámetros correspondientes y forzando una nueva actualización.

# Lecciones aprendidas

Siempre validar primero conectividad y registros antes de asumir una falla del producto.
